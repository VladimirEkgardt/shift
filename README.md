# Filter Utility
Утилита для фильтрации данных по типам из текстовых файлов.

## Функциональность
- Фильтрация целых чисел, вещественных чисел и строк
- Запись в отдельные файлы с возможностью определить путь
- Статистика обработки (краткая и полная)
- Гибкие настройки вывода

## Опции
- -o путь для выходных файлов. Пример: -o [path]
- -p префикс имен файлов. Пример: -p [prefix]
- -a режим добавления
- -s краткая статистика
- -f полная статистика

## Особенности реализации
- Автоматическое создание выходных путей
- Некорректные аргументы: сообщения об ошибках
- При чтении несуществующего файла программа не падает, продолжает обрабатывать следующие файлы, если такие есть
- При одновременном указании опций -s и -f, приоритет отдается -f из-за дублирования данных
- После опции -o должен быть написан путь для файла. Иначе будет выводиться ошибка. 
Обработка продолжиться, выводиться будет в текущую директорию 
- После опции -p должен быть написан префикс для файла. Иначе будет выводиться ошибка. 
Обработка продолжиться, будут созданы файлы без префикса
- Префикс не может начинаться с "-"
- Путь не может начинаться с "-"
- Java 17+
- Система сборки: gradle 8.5

## Входные данные
- Вещественные числа: разделитель - точка(пример, 2.3)
- Целые числа могут быть любой длины
- Каждая строка входных данных файлов содержит одно значение
- Пустые строки считаются строками

## Формат вывода
### Имена файлов по умолчанию:
- "integers.txt" - целые числа
- "floats.txt" - вещественные числа
- "strings.txt" - строки

## Запуск утилиты
    java -jar shift-1.0.jar [options] file1.txt file2.txt
