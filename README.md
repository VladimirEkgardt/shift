# Filter Utility
Утилита для фильтрации данных по типам из текстовых файлов.

## Функциональность
- Фильтрация целых чисел, вещественных чисел и строк
- Запись в отдельные файлы с возможностью определить путь
- Статистика обработки (краткая и полная)
- Гибкие настройки вывода

## Опции
- -o путь для выходных файлов. Пример: -o [path]
- -p префикс имен файлов. Пример: -p [prefix]
- -a режим добавления
- -s краткая статистика
- -f полная статистика

## Особенности реализации
- Вещественные числа: разделитель - точка(пример, 2.3)
- Автоматическое создание выходных путей
- Некорректные аргументы: сообщения об ошибках
- При чтении несуществующего файла программа не падает, продолжает обрабатывать следующие файлы, если такие есть
- Java 17+
- Система сборки: gradle 8.5

## Запуск утилиты
    java -jar shift-1.0.jar [options] file1.txt file2.txt
